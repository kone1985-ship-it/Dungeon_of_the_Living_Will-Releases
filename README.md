Dungeon of the Living Will（配布用）

# はじめに
* 配布ZIPを解凍したフォルダで作業してください。
* 「DungeonTools.exe」を起動してください。
* 「設定」ページで「APIキー」「モデル名」「Base URL」「画像生成モデル(任意)」を入力してください。推奨設定は下記のとおりです。
    - APIキー： 下記で発行したもの
        - [https://console.x.ai/home](https://console.x.ai/home) から sign in もしくは create account する
        - console が開くので、左のメニューから「API Keys」を選択
        - 適当な名前を入力して、create API key を押す
        - API key が発行されるので、それを「APIキー」に入力。
        - 左のメニューから Dashboard を選択し Puerchase をクリック もしくは画面上部に「You have no paid credits」と出るのでクリック
        - 課金額を入力。最初は $5 で十分。
        - 「Add your first card」 を押して支払情報を登録
    - モデル名： grok-4-1-fast-reasoning
    - Base URL： https://api.x.ai/v1
    - 画像生成モデル定義： grok-2-image-1212
    - 最後に「保存」をクリック
* 「Dungeon_of_the_Living_Will.exe」を起動してください。

※「Dungeon_of_the_Living_Will.exe」と同じ場所にある「_internal」フォルダは必要です（削除/移動しないでください）。

# 遊び方(画面説明)
## キャラクター選択
ダンジョンに潜るキャラクターを選択します。最大3人です。選んだら「迷宮」を押してください。

## ダンジョン選択
潜るダンジョンを選択(A/Dキー)します。
「詳しく見る」で依頼がAIにより生成されます。
「このダンジョンで開始」でダンジョンに潜ります。

## ダンジョン探索
ダンジョンを探索します。
W/A/S/D で移動、左側のコメント入力欄や選択肢で行動を指定できます。
行動に応じてAIがイベントを生成したり、ランダムで敵と戦闘になったりします。
通常はダンジョン深部にいるボスを倒すとクエストクリアです。

## キャラクター詳細（右クリック）
キャラクター画像の上で右クリックすると詳細画面が開きます。
- 詳細画面を開いている間は、誤操作防止のため背景の操作（移動・クリック等）は無効になります
- クリック / 右クリック / ESC で閉じます（ホイールでスクロールできます）

## キャラクター同士の対戦
キャラクター選択画面から、味方パーティ vs 敵パーティの対戦を開始できます。
- キャラクター選択で味方を1〜3人選ぶ
- 右下の「対戦」を押す
- 敵選択画面で敵を1〜3人選ぶ
- 右下の「開始」を押す

## 報酬ポイント（RP）
画面右上に RP（報酬ポイント）が表示されます。

- 宝箱・アイテムなど財宝を手に入れたときに少し増えます
- 依頼攻略（クエストクリア）で増えます

RPの使い道（最大HP/MP/SPの強化）
- キャラクター詳細画面に、RPを消費して最大値を+1するボタンがあります。
- HP / MP / SP を +1 できます
- 消費RPは「強化後の最大値」と同じです（例：最大HPが10→11なら必要RPは11）

## キャラクター作成
イラスト(2:3の縦長)からキャラクターを作成します。
同梱の「DungeonTools.exe」から起動します。
イラストを選んだら、名前やタグ、プロフィール、HP/MP/SP、スキルを入力してください。面倒ならAIに作成をお願いできます。

保存先（ユーザデータ）:
- `%USERPROFILE%\AppData\LocalLow\Dungeon_of_the_Living_Will\characters`

## ダンジョンの作成
同梱の「DungeonTools.exe」から作成できます。

- DungeonTools の左側「Pages」から「ダンジョン」を開く
- ダンジョン名・画像・各種設定を入力する
- 「保存」→「Save」で保存する（保存形式は「フォルダ」「ZIP」どちらでも可）

保存先（ユーザデータ）:
- `%USERPROFILE%\AppData\LocalLow\Dungeon_of_the_Living_Will\dungeons`

作成したダンジョンは、ゲームの「ダンジョン選択」に表示されます。

## 独自の作劇命令の作成
AI-GMの振る舞いに追加の指示（文体・テンポ・禁止事項など）を与える「カスタムインストラクション」を作成できます。

作成（DungeonTools）
- DungeonTools の左側「Pages」から「カスタムインストラクション」を開く
- 「名称」「簡単な説明」「本体」を入力して「保存」

保存先（ユーザデータ）:
- `%USERPROFILE%\AppData\LocalLow\Dungeon_of_the_Living_Will\instructions` 配下の `*.json`

ゲームで使う（キャラクター選択）
- 右下の大きいボタン（初期表示は「デフォルト作劇」）をクリックして、使用する作劇命令を切り替えます
- 選んだ状態で「迷宮」または「対戦」を開始すると、その作劇命令がAI-GMに適用されます
